# MeshCoreStat
Кроссплатформенное консольное приложение для получения детальной статистики работы узла MeshCore по COM‑порту.

_English version: see [README.MD](README.MD)._ 

## Описание

MeshCoreStat подключается к узлу MeshCore по указанному COM‑порту и последовательно выполняет набор [CLI‑команд](https://github.com/meshcore-dev/MeshCore/blob/main/docs/cli_commands.md):
`ver`, `board`, `get name`, `stats-core`, `stats-radio`, `stats-packets`.

Ответы на эти команды парсятся и отображаются в виде «дашборда» в терминале. Интерфейс одинаков как для разового запуска, так и для режима автообновления:

- **Node**: имя узла (`get name`), плата (`board`), версия прошивки MeshCore (`ver`), параметры подключения (порт, скорость, интервал опроса).
- **Core**: напряжение питания, аптайм в человекочитаемом виде и в секундах, счётчик ошибок, длина очереди.
- **Radio**: уровень шума, последний RSSI/SNR, суммарное время работы передатчика/приёмника и их доля от аптайма в процентах.
- **Packets (total)**: накопленные счётчики пакетов (`recv`, `sent`, `flood_*`, `direct_*`, `recv_errors`).
- **Relative (per hour)**: те же счётчики, пересчитанные в «пакетов в час» с учётом текущего аптайма.

Приложение оптимизировано под минимальное потребление ресурсов (small footprint) и может использоваться на слабых машинах и одноплатных компьютерах.

## Параметры командной строки

Приложение принимает следующие параметры:

- **Порт (обязательный)**: имя COM‑порта, к которому подключён узел MeshCore (например, `COM12` или `/dev/ttyUSB0`).
- **Скорость (baud rate, необязательный)**: скорость соединения, по умолчанию `115200`.
- **Период автообновления, сек (необязательный)**: интервал между опросами. Значение `0` (по умолчанию) даёт один снимок и немедленный выход, вывод остаётся в терминале; значение `> 0` включает автообновление.

Точный формат запуска и перечень доступных опций смотрите в встроенной справке приложения (`--help`).

## Типичные сценарии использования

- **Быстрая проверка узла**:  
  `meshcorestat --port COM12`  
  Один раз выводит дашборд (Node/Core/Radio/Packets) и завершает работу, не очищая экран.

- **Мониторинг в реальном времени**:  
  `meshcorestat --port COM12 --interval 2`  
  Каждые 2 секунды обновляет показания (в том числе относительную статистику в пакетов/час); выход по клавишам `q` или `Esc`.
